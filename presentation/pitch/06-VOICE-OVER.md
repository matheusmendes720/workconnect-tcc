> **Arquivo de produção de áudio.** Este é o texto literal que será gravado
> (humano ou TTS como ElevenLabs). Cada bloco tem TC (timecode), cena do
> `teaser.html`, texto exato da fala, e dicas de entonação.
>
> **Métricas-alvo:** 2:00 totais, ~270 palavras, ~135 palavras/minuto
> (ritmo profissional, com pausas de 200–400 ms antes das revelações numéricas).

---

## 🎙️ Convenções de leitura

- `|` indica pausa de 200 ms (entre palavras).
- `||` indica pausa de 400 ms (antes de revelação numérica ou ênfase).
- `[ênfase]` indica a sílaba/ palavra a stressing na leitura.
- **Tom geral:** sóbrio, confiante, sem jargão. "Você" (não "o usuário").
- **Velocidade:** ~135 wpm. Mais lento que conversa normal; dá ar de "narração de documentário".
- **Microfone:** condenser cardioid, gain -6 dB, 48 kHz/24-bit.
- **Loudness alvo:** -16 LUFS integrated, true peak -1.5 dBTP.

## 🎬 Sugestões de produção

### Locução humana (recomendado)

```bash
# Audacity (free) + microfone USB (ex: Blue Yeti, Fifine K669)
# Gravar em 48 kHz / 24-bit
# Aplicar: noise reduction, EQ (cortar graves <80Hz, leve boost 2-4kHz), compressão 4:1
# Exportar: WAV 48 kHz 24-bit
# Converter para MP3 192 kbps 48 kHz: ffmpeg -i voice.wav -b:a 192k voice.mp3
# Aplicar loudnorm I=-16 TP=-1.5 LRA=11
```

### Locução TTS (rápido, qualidade aceitável)

```bash
# ElevenLabs free tier (10k chars/mês)
# Voz sugerida: "Antoni" (PT-BR, masculina, narrativa)
# Ou Azure Cognitive Services (neuronal TTS, 500k chars/mês free)
# Ou Google Cloud TTS (1M chars/mês free, vozes Studio)
```

### Mix final com trilha

```bash
# Trilha: Artlist ou Pixabay Music
# Estilo: cinematic ambient, ~75 BPM, sem vocais
# Volume da trilha: -18 dB sob a narração (afade in/out 1s/5s)
# Normalizar tudo para -16 LUFS integrated (loudnorm)
```
